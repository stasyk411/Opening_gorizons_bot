# Открытые горизонты (@Opening_gorizons_bot)

Психологический навигатор для работы с эмоциями, убеждениями и сложными жизненными ситуациями.

## 🔒 Конфиденциальность и безопасность данных

- Все диалоги хранятся **локально** в базе данных SQLite (`bot_data.db`).
- База данных пользователей **исключена из репозитория** и никогда не передаётся третьим лицам.
- Методологическое ядро, системные промпты и конфигурация являются приватными и не входят в публичный код.
- Данный репозиторий содержит только технический каркас бота.

## ⚠️ Важное предупреждение

Бот является инструментом самопомощи и **не заменяет** квалифицированного специалиста (психолога, психиатра).  
При острых кризисных состояниях, угрозе жизни или здоровью немедленно обращайтесь в экстренные службы:

- 📞 103 — Скорая помощь
- 📞 112 — Единый номер экстренных служб
- 📞 8-800-2000-122 — Телефон доверия (Россия)

## 🛠 Технический стек

- Python 3.10+
- pyTelegramBotAPI
- OpenRouter API
- SQLite

## 📋 Требования для запуска

Для работы бота необходим файл `.env` со следующими переменными (значения не включены в репозиторий):

- `TELEGRAM_BOT_TOKEN`
- `OPENROUTER_API_KEY`
- `OWNER_ID`
- `MODEL_NAME`
