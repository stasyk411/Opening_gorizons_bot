# Черновик языкового лексикона (Language Engine)

> **Статус:** Черновик. Не внедрён в код.
> **Источник:** Фразы выписаны из реальных тестовых диалогов.
> **Правило:** Добавлять только пары, подтверждённые поведением пользователей. Не придумывать сверху.

## Таблица перевода: Внутренний концепт → Человеческий язык

| Внутренний концепт (Движок) | Человеческие формулировки (Из диалогов)                                                                            | Источник / Контекст                                                      |
| :-------------------------- | :----------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------- |
| Позиция «Хороший»           | «Не хочу выглядеть жадным», «Не хочу давить», «Важнее не испортить отношения»                                      | Диалог о долге: клиент выбирал между возвратом денег и сохранением связи |
| Позиция «Невидимка»         | «Отстраняюсь, чтобы не тратить силы», «Мысль всё равно возвращается», «Проще промолчать»                           | Тот же диалог: клиент описывал внутреннее трение                         |
| Интерпретация               | «Висит в воздухе», «Недоверие к продукту или к себе», «Замёл ситуацию»                                             | Клиент описывал неопределённость с долгом и сотрудничеством              |
| Убеждение / Правило         | «Никакого правила не сформулировалось», «Не пытаюсь повесить ярлык», «Хочу решить проблему, а не доказать правоту» | Клиент отказался от обобщения, остался в прагматике                      |

## Принцип заполнения

- **1:N** — один концепт может иметь множество человеческих формулировок.
- **Контекст важен** — фраза без указания источника/ситуации не добавляется.
- **Только из реальности** — запрещены фразы, которые пользователь никогда не произносил.
